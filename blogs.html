Question-3: What is event delegation in the DOM?

Event delegation is a define where you attach a single event listener to a parent element and it will handle events triggered on its child elements ,alternatively of adding separate listeners to each child element.

Question-4: What is event bubbling in the DOM?

Event bubbling is a refers to the behavior where and when an event occurs on an element, it not only happens on that element but also on its parent elements in tne hierarchy.
